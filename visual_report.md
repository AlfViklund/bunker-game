# Сводный визуальный отчет по редизайну системы «Бункер 2077»

Этот отчет содержит краткое описание новой дизайн-системы, реализованной на основе стандартных палитр Tailwind CSS v4 и Radix UI, а также обоснование отказа от неонизированных шаблонов в пользу тактического милитари-минимализма (Tactical Post-Apocalyptic UI).

---

## 1. Сравнительный анализ: До и После

| Элемент | Прежний дизайн (Cyberpunk Neon) | Новый дизайн (Tactical Post-Apoc UI) | Обоснование изменений |
| :--- | :--- | :--- | :--- |
| **Философия** | Хаотичный киберпанк, переизбыток неоновых свечений, глитчи на кнопках, "премиум ИИ-хайп". | Строгий военный минимализм, эргономичность терминала жизнеобеспечения, физический износ. | Снижение зрительной усталости, концентрация на геймплее и текстовых дебатах, повышение реализма. |
| **Фон** | Глубокий неоново-синий / черный (`#05050a` с неоновыми тенями). | Чистый индустриальный темный (`zinc-950` #09090b, панели `zinc-900` #18181b). | Настоящие матовые текстуры без свечений повышают контрастность информационных блоков. |
| **Акценты успешности**| Токсичный зеленый (`#00ff66` с эффектом neon-glow). | Тактический изумрудный (`emerald-500` #10b981 / `emerald-400` #34d399). | Соответствие WCAG 2.1 AA по читаемости и контрастности (более 4.5:1), отсутствие размытия букв. |
| **Предупреждения** | Желтый неон (`#ffe600`). | Военный янтарный (`amber-500` #f59e0b / `amber-400` #fbbf24). | Приглушенный теплый спектр, традиционно применяемый в промышленных интерфейсах. |
| **Критический статус** | Маджента / Розовый неон (`#ff007f`). | Радиационный багровый (`rose-600` #e11d48 / `rose-500` #f43f5e). | Ассоциация с угрозой радиации, четкое выделение фазы опасности и действия "изгнать". |
| **Типографика** | Декоративный Orbitron для заголовков. | Универсальный `Inter` для UI; `JetBrains Mono` для моноширинных данных. | Устранение ломаных линий Orbitron, мешавших беглому восприятию элементов управления. |
| **Интерактив** | Кибер-клики со звуком глитчей, светящиеся тени при ховере. | Механические реле и приглушенные щелчки, жесткие обводки `ring-2 ring-sky-500` при фокусе. | Полноценная тактильная имитация старого оборудования, проработанный фокус клавиатуры (A11y). |

## 2. Разбор Цветовой Системы (Tailwind v4)

```css
/* Базовые поверхности */
--bg-primary:    theme(colors.zinc.950); /* #09090b - Основной фон экрана */
--bg-surface:    theme(colors.zinc.900); /* #18181b - Подложка плашек */
--bg-interactive:theme(colors.zinc.800); /* #27272a - Второстепенные контролы, кнопки */
--color-border:  theme(colors.zinc.800); /* #27272a - Микро-границы */

/* Тактические статусы */
--status-safe:   theme(colors.emerald.500); /* #10b981 - Системы в норме, игрок готов */
--status-warn:   theme(colors.amber.500);   /* #f59e0b - Деградация ресурсов, CynicBot */
--status-danger: theme(colors.rose.600);    /* #e11d48 - Критический таймер, PanicBot, Изгнание */
--status-info:   theme(colors.sky.500);     /* #0ea5e9 - Навигация по картам, StrategistBot */

/* Типографика */
--text-primary:  theme(colors.zinc.100); /* #f4f4f5 - Максимально разборчивый текст */
--text-muted:    theme(colors.zinc.400); /* #a1a1aa - Вспомогательные данные */
```

## 3. Ключевые интерактивные компоненты

1.  **Hold-to-Confirm Button (Кнопка изгнания):**
    Красные кнопки со статусом повышенного риска требуют удержания мыши в течение 1.5 секунд. Визуально это выражается в заполнении кнопки градиентом от `rose-950/40` к `rose-600` слева направо. При отжатии кнопки мыши таймер сбрасывается. Это защищает от случайных вылетов и повышает драматический эффект принятия решения.

2.  **Typewriter Log Interface (Консоль Финала):**
    Итоговый бортовой журнал выводится посимвольно в окне консоли со шрифтом `Fira Code` / `JetBrains Mono` под аккомпанемент тикающих звуков реле. Игроку предоставлена системная кнопка «[ SKIP LOG EFFECT ]» для мгновенного раскрытия всего текста истории.

3.  **Survivor Cards (Карты Выжившего):**
    7 карт характеристик выровнены по сетке и скрыты стальными заглушками. При развороте используется быстрый 3D flip-effect без инерционных колебаний, что гарантирует моментальный отклик интерфейса.

## 4. Карта доступности (A11y Map)

*   **Клавиатурный доступ:** Поддерживается полный цикл навигации по `Tab`. Фокус-стейты четко выражены рамкой `ring-2 ring-sky-500` на темном фоне.
*   **Контрастность текстов:** Все тексты соответствуют требованиям WCAG 2.1 AA. Коэффициент контрастности основного текста превышает 10:1.
*   **Семантическое разделение информации:** Использование иконок рядом с критическими метками (значки `⚠️`, `❓`, `☠️`), чтобы цвет не являлся единственным каналом передачи информации.
