# Документ Проекта: «Бункер 2077» — Тактический Симулятор Выживания

## 1. Обзор Проекта
**«Бункер 2077»** — дискуссионная психологическая Party-игра в атмосфере постиндустриального апокалипсиса. Игра адаптирована как для одиночного прохождения (в режиме симуляции с ИИ-агентами), так и для мультиплеерных сессий (до 10 пользователей в одной комнате). Основная цель игроков — договориться о распределении ограниченных ресурсов убежища и путем голосования определить состав выживших, способных пережить внешнюю катастрофу.

---

## 2. Основные Игровые Механики и Фичи

1. **Динамическая генерация сценариев катастроф:** 
   С помощью интеграции API Pollinations.ai (LLM) генерируется уникальный вводный контекст катастрофы (солнечный супершторм, отказ биореактора, техногенная мутация вирусов). Визуализация катастрофы строится на генеративных изображениях в приземленном индустриально-милитаристском стиле (без неонового перенасыщения).
   
2. **Интерактивные ИИ-Агенты (Боты):**
   Боты функционируют на базе предопределенных поведенческих моделей (психотипов):
   *   *Паникер (PanicBot):* импульсивное поведение, высокая чувствительность к базовым ресурсам бункера, призывы исключать ослабленных игроков.
   *   *Циник (CynicalBot):* прагматичная оценка полезности каждого выжившего на основе сухих цифр и характеристик, агрессивная критика нематериальных навыков.
   *   *Стратег (StrategistBot):* поиск логических связей, кооперация, оценка долгосрочного влияния профессий и здоровья спасенных на выживаемость.
   Боты активно участвуют в текстовом чате дебатов, оперируют фактами своих карт и карт других игроков, а также голосуют в реальном времени.

3. **Индустриальная Система Авторизации и Связи:**
   *   **Fast Guest Auth:** Однокликовый гостевой вход. Генерируется уникальное системное имя и UUID, которые сохраняются в `localStorage` и регистрируются через Supabase Anonymous Auth. Прогресс и история профиля защищены без необходимости ввода почты.
   *   **Comm-Link (Список друзей):** Быстрое добавление друзей по уникальному идентификатору (ID) игрока. Статусы подключения («Свободен», «В игре», «Офлайн») синхронизируются через Supabase Realtime Broadcast.
   *   **Direct Invite:** Мгновенный вызов в лобби через отправку сигналов внутри Websocket-каналов без задержек.

4. **Генерация Эпилога (Бортовой Журнал):**
   По окончании финального раунда голосования система анализирует список выживших, их профессии, состояние здоровья, багаж и черты характера. На базе этих данных ИИ моделирует детальный отчет выживания группы на 10 лет вперед, описывая внутренние конфликты, распределение ресурсов и итоговый статус миссии.

## 3. Визуальная Философия и Интерфейс (Tactical Post-Apocalyptic UI)

В отличие от стандартных шаблонов «неонового киберпанка», визуальный стиль проекта построен на принципах приземленной индустриальности:
*   **Палитра:** Доминирование темных стальных оттенков (`zinc-950`, `zinc-900`) с контрастными индикаторами тактического цвета: зеленый (`emerald-500`) — стабильность/готовность, янтарный (`amber-500`) — регулировка систем/предупреждения, багровый (`rose-600`) — критическая опасность/изгнание.
*   **Типографика:** Сочетание читаемого шрифта `Inter` для общих интерфейсных блоков управления и моноширинного `JetBrains Mono` для чтения чата, вывода результатов голосования и системных данных.
*   **Компоненты:** Применение чистых матовых плашек, полупрозрачного стекла с минимальной размытостью, микро-границ (`border-zinc-800`), четких прямоугольных кнопок и жестких фокусов ввода клавиатуры.

## 4. Архитектура и Технический Стек

*   **Фреймворк:** Next.js 16 (App Router) с полноценной поддержкой React 19 (Server Actions, оптимизированные хуки состояний).
*   **Стилизация:** Tailwind CSS v4 с нативной компиляцией, сокращенным размером CSS-бандла и поддержкой современных CSS-переменных.
*   **Анимации:** Framer Motion (плавное раскрытие карт, жесткие переходы сдвижных панелей, системные анимированные шкалы прогресса).
*   **Бэкенд-инфраструктура:** Supabase (PostgreSQL для хранения игровых сессий и профилей пользователей, Row Level Security для защиты данных лобби, Realtime Subscriptions для мгновенной синхронизации ходов и чата).
*   **Интеграция ИИ:** API Pollinations.ai (текстовые модели для ведения логов, дебатов ботов и генерации катастроф; диффузионные модели для генерации индустриальных артов катастроф).
*   **Хостинг & CI/CD:** Vercel с автоматическими превью-деплоями при push-событиях в репозиторий GitHub.
