# AGENTS.md — Bunker Tactical Post-Apocalyptic Party Game

Этот документ служит кратким руководством для разработчиков и автономных ИИ-агентов, работающих над кодовой базой проекта «Бункер 2077».

## 1. Запуск и Сборка Проекта
*   Сборка приложения: `pnpm build`
*   Статическая проверка типов: `pnpm tsc --noEmit`
*   Запуск в режиме разработки: `pnpm dev`

## 2. Архитектура и Соглашения
*   **Стек:** Next.js 16 (App Router), React 19, Tailwind CSS v4, Supabase (Auth, RLS, Realtime), Pollinations.ai (AI-агенты и генерация мета-данных).
*   **Визуальная тема:** Tactical Post-Apocalyptic UI. Исключить неоновые киберпанк-эффекты и глитчи. Придерживаться строгой индустриальной эстетики:
    *   *Фон:* глубокий матовый черный (`bg-zinc-950` / `#09090b`), панели (`bg-zinc-900` / `#18181b`), обводка (`border-zinc-800` / `#27272a`).
    *   *Тактическая безопасность:* зеленый (`emerald-500` / `#10b981`, `emerald-400` / `#34d399`) — для статуса готовности, подключения и успешных системных тестов.
    *   *Предупреждения:* янтарный (`amber-500` / `#f59e0b`, `amber-400` / `#fbbf24`) — для шкал износа систем, настроек сложности и CynicBot.
    *   *Опасность:* багровый (`rose-600` / `#e11d48`, `rose-500` / `#f43f5e`) — критические таймеры, кнопки изгнания в пустошь и PanicBot.
    *   *Инфо-Акцент:* приглушенный голубой (`sky-500` / `#0ea5e9`, `slate-400` / `#94a3b8`) — для раскрытия активных карт, выделенных карточек и StrategistBot.
*   **Шрифты:** интерфейсы на базе шрифта `Inter` (sans-serif), текстовые чаты, цифры и логи на `JetBrains Mono` (monospace).
*   **Синхронизация в реальном времени:** Согласование игрового состояния, мгновенный обмен сообщениями чата и действия ботов выполняются через каналы Supabase Realtime Broadcast и подписки на таблицы PostgreSQL.
*   **Функциональные возможности:** Вход в однокликовый гостевой режим (UUID в LocalStorage) с бесшовной интеграцией Supabase Anonymous Auth, система связи (Comm-Link) с отправкой инвайтов, интеграция с Pollinations LLM для ведения обсуждений ИИ-ботами и диффузионных сетей для артов катастроф.
